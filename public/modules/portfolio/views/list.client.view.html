Dissertation: ScatteredWorld

For my 3rd year University dissertation I created a 2D distributed virtual world in Java. This project recursively split the world up into smaller and smaller squares based on load and assigned them to peers. A DHT (Chord) was used for region lookup.

The project was successful in demonstrating that the methods used were feasble for real word usage and that offloading regions would significantly reduce the load for each individual node.

I have included a download for the GUI jar, however it may not be clear what is going on when you open it. In fact I expect it to be majorly confusing unless you are me. I plan to get the source up onto GitHub with an explanation soon.

BBQ Source

BBQ Source was a group University assignment using Google App Engines's Python integration. It was designed as a collaborative code editor, one may sign up with their Google account create a project, create a group, assign groups to projects etc. We used TogetherJS to allow real time collaboration.

This was a fun project though it made me learn a couple of lessons about JavaScript and TogetherJS the hard way.

  1. You can't run two instances of TogetherJS on the same page.
  2. An IFrame does NOT count as a new page

For these two reasons, collaboration is very wonky.
